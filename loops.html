For loop

Definition : For loop in Javascript is the most commonly used loop to repeat a block of code a specific num of times
syntax: for(initialization ; condition ; increment / decrement) { //code to be executed }

Initialization -> Run once(start value)
condition -> checked before each iteration
increment / decrement -> update loop variable

While loop

Definition : the while loop executed the set of statement as long as the condition is true.
condition checked before executed
If condition is false initially -> loop won't run
syntax: while(expression){ statement to be executed }
the length is unknown it will run as long as it is false

do while loop

Execute at least one time even if the condition is false
syntax:
do{ statement to be executed
increment
}while (expression){ }

For-in loop

The For-in is used to loop through object properties
In each iteration, one property from object is assigned to a variablename and this loop continues till all the properties of object are exhausted
syntax :
for (variable name in object){ }
